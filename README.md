# CURD-Operation-Flutter-Firebase-Complete-App

في هذه الفيديو ، ستتعلم اضافة Cloud Firestore مع Flutter وانشاء تطبيق من خلالة ستفهم عمليات التعامل مع البيانات وتخزينها في قاعدة الفايربيز  وهذا النوع من التطبيقات  يسمى  CRUD operations وتعني هذه الاحرف المختصره  (create, read, update and delete) , من خلال هذه العمليات الاربع سيصبح عندك القدرة في التحكم في تخزين البيانات وادارتها بشكل صحيح.
